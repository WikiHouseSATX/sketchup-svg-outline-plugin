This is a stripped down version of the original Sketup SVG outline Plugin from Simon Beard.

His original version was built to be used directly as extension in SketchUp.  That version is available
at : Website: http://simonbeard.github.io/sketchup-svg-outline-plugin/

     Plugin Download: https://github.com/simonbeard/sketchup-svg-outline-plugin/releases/download/0.9999999/flightsofideas.rbz


This version is designed to be used by other extensions to output SVGs.
